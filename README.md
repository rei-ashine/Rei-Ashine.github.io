# [Rei-Ashine](https://rei-ashine.github.io)

"The book called Life doesn't have the answers written at the back."

## Tech Stack

- **Framework**: [Hugo](https://gohugo.io) (Static Site Generator)
- **Hosting & CI/CD**: [GitHub Pages](https://pages.github.com) via GitHub Actions
- **Analytics**: Google Tag Manager (GTM)
- **Theme**: Modified version of [m10c](https://github.com/vaga/hugo-theme-m10c)

## Local Development

To run this website locally on your machine:

1. Ensure you have [Hugo (Extended version)](https://gohugo.io/installation/) installed.
1. Clone this repository.
1. Run the local development server:

```bash
hugo server -D
```

1. Open your browser and navigate to `http://localhost:1313/`.
